# Kids Agency Website

A comprehensive digital experience designed for a creative agency specializing in children's content. The project focused on creating an engaging, colorful, and user-friendly interface that appeals to both children and their parents.

## Key Features

- Interactive animations and micro-interactions
- Responsive design optimized for all devices
- Accessibility-first approach
- Performance-optimized for fast loading

## Technologies Used

- Next.js
- OGL (raw) for WebGL planes
- GSAP for timelines
- Vanilla CSS (PostCSS)
